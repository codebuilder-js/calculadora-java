# Calculadora Java

Projeto simples de calculadora feito em Java no terminal.

O programa realiza operações matemáticas básicas e algumas operações extras, como potência, raiz quadrada e porcentagem.

## Tecnologias usadas

- Java
- Scanner
- ArrayList
- IntelliJ IDEA / Eclipse

## O que o projeto faz

A calculadora consegue:

- Somar números
- Subtrair números
- Multiplicar números
- Dividir números
- Calcular potência
- Calcular raiz quadrada
- Calcular porcentagem
- Mostrar histórico de operações
- Validar divisão por zero
- Validar entradas inválidas
- Fazer autenticação de usuário
- Limitar quantidade de operações

O usuário escolhe uma opção no menu e informa os números pelo terminal.

## Como executar

1. Instale o Java na sua máquina
2. Compile o arquivo:

```bash
javac Main.java
```

3. Execute o programa:

```bash
java Main
```
